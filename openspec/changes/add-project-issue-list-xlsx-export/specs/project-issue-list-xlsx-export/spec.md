### Requirement: Export SHALL download the current project issue list result set

When a user triggers XLSX export from the project issue list toolbar, the system SHALL generate a file for the current project issue list result set, regardless of the active layout, using the currently active filters applied to that page.

#### Scenario: Export respects active filters

- **WHEN** the project issue list has active filters that reduce the visible result set
- **THEN** the exported workbook SHALL contain only issues matching the currently active filters

#### Scenario: Export works from any supported layout

- **WHEN** the user triggers `–≠–∫—Å–ø–æ—Ä—Ç –≤ xlsx` while the project issue list is in `list`, `kanban`, `spreadsheet`, `calendar`, or `gantt_chart` layout

### Requirement: Exported workbook SHALL include filter summary and required columns

The exported workbook SHALL begin with summary information about the active filters and SHALL then include a tabular section with columns determined by the current layout and display settings.
